FilterPlugin

This Plugin simply hooks up an input field to filter through an array of results, and display those results on the page.

$('#myInputField).filterPluing(arrayToFilter [,options]);

arrayToFiler must be an array of string literals. options must be an object, containing zero or more properties from the following:

{
    caseSenstive: true, // Default is true
    wholeWords: false // Default is false
}
